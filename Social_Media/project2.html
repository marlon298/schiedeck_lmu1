Communication between computer and user

What happens when you type an URL in a browser:

  • type URL
  • Browser looks up IP in cache -> DNS CACHE
  • Browser looks up IP using recursive DNS Lookup
  • Browser establishes TCP connection with server
  • Browser sends a http to server
  • Server sends back http response
  • Browser renders http connection

Client

software application, a web browser or any device that initiates a request to the server

Server

responsible for processing and fulfilling client requests

Domain Name

Allows users to access websites using memorable names instead of IP adresses

HTTP Methods

  • POST
  • GET
  • PUT
  • DELETE
