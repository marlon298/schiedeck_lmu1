Frontend Development

HTML

Website structure

Definition: HTML is the standard markup language for creating and structuring web pages. It uses a system of elements and tags to define the various parts of a document, such as headings, paragraphs, images, links, and more.

Role in Frontend Development: HTML provides the basic structure and content of a webpage. It defines the elements that browsers use to render the page, creating the foundation for web development.

CSS

User interface

Definition: CSS is a style sheet language used for describing the presentation of a document written in HTML. It controls the layout, design, and appearance of web pages, allowing developers to style and format HTML elements.

Role in Frontend Development: CSS is crucial for styling and formatting web content. It enables the separation of structure (defined by HTML) and presentation (defined by CSS), providing a flexible and maintainable way to design web interfaces.

JavaScript

Interactions on website

Definition: JavaScript is a high-level, versatile programming language primarily known for its role in building dynamic and interactive web pages. It is supported by all major browsers and allows developers to add behavior, manipulate the DOM (Document Object Model), and interact with users.

Role in Frontend Development: JavaScript is a key language for frontend development. It enables the creation of responsive and interactive user interfaces, handling events, making asynchronous requests, and dynamically updating content on web pages. It plays a crucial role in creating rich, client-side experiences.

JSON

Connection to server, translator human and computer language

Definition: JSON is a lightweight data interchange format that is easy for humans to read and write and easy for machines to parse and generate. It is often used to transmit data between a server and a web application, as well as for configuration files.

Role in Frontend Development: JSON is commonly used to structure and exchange data between a web server and a web application. It is easy to parse in JavaScript, making it a popular format for APIs (Application Programming Interfaces) and data storage.
